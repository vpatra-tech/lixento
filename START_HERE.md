## 🚀 Quick Start (3 Steps)

### 1. Open Website
```
Double-click: index.html
OR
Run: python -m http.server 8000
Then visit: http://localhost:8000
```

### 2. Customize
Edit these files:
- **Change company name:** Find & replace "ColorTech" → Your name
- **Change brand color:** Edit `css/theme.css` line with `#dc3545`
- **Update contact info:** Edit `pages/contact.html` and `index.html` footer

### 3. Deploy
Upload all files to hosting provider:
- Netlify (recommended - auto-deploy)
- GitHub Pages (free)
- Shared hosting (GoDaddy, Bluehost)
- Cloud (AWS, Google Cloud, Azure)

## 🎨 Customization (5 Minutes)

### Change Company Name
```
Find: ColorTech
Replace: YourCompanyName
Files: All HTML files
```

### Change Brand Color
```
Edit: css/theme.css
Find: --ct-primary: #dc3545
Change to: Your color code
```

### Update Contact Info
```
Files: index.html, pages/contact.html
Find: +1-800-PAINT-01
Replace: Your phone number

Find: sales@colortech.example.com
Replace: Your email
```

### Add Your Logo
```
1. Save logo as: assets/images/logo.png
2. Update navbar brand in HTML:
   <img src="assets/images/logo.png" alt="Logo">
```
